## 5. 调整浪之平台形

### 5.1 平台形验证手册

以下6类规则或指引，按照重要程度排序。手册中的"价格"指的是"某个浪型的起点到终点之间的长度"，"运行总量"指的是"某个浪型从所有子浪的最低点到所有子浪的最高点之间的长度"。"价格运动百分比"指的是"某个浪型的所有子浪的最高点超过所有子浪的最低点的百分比"。

**平台形的浪型规则**
1) 由3个子浪构成，标注为a、b、c。
2) 这3个子浪的内部结构：
   ① a浪必须是除收缩三角形外的调整浪。
   ② b浪必须是除收缩三角形外的调整浪。
   ③ c浪必须是推动浪或终结楔形。

**平台形的比率规则**
1) b浪的"运行总量"必须回撤超过a浪的"运行总量"的70%。
2) b浪的"运行总量"必须小于a浪的"运行总量"的2倍。
3) c浪必须与a浪有重叠。
4) c浪的"运行总量"必须小于等于a浪的"运行总量"和b浪的"运行总量"中最长一个的2倍，并且小于等于a浪的"价格"的3倍。

**平台形的时间规则**
1) a浪的最小时间没有限制。
2) b浪的最小时间没有限制，最长时间不能超过a浪的时间的10倍。
3) c浪的时间不能超过a浪的时间的10倍。
4) c浪的时间不能超过b浪的时间的10倍。

**平台形的浪型指引**
1) a浪通常是锯齿类（单锯齿、双锯齿、三锯齿之一）。
2) b浪通常是锯齿类（单锯齿、双锯齿、三锯齿之一）。
3) b浪极少是平台形。
4) c浪极少是终结楔形。
5) c浪经常约等于a浪和b浪的长度。
6) a浪中最大的子浪的价格通常小于a浪的价格。
7) b浪中最大的子浪的价格通常小于a浪的"价格"。

**平台形的比率指引**
1) b浪的"价格"通常大于a浪的"价格"的0.95倍，小于a浪的"价格"的1.4倍。
2) c浪经常超过b浪的1倍。不经常超过a浪的1.4倍。不经常超过b浪的1.4倍。
3) c浪的比率按照常见程度排序: c=a*1, c=a的终点, c=a*1.618, c浪=a的终点算起向外扩展0.618*a。
4) 如果c浪长于b浪，那么c浪经常达到从a浪终点起向外扩展a*61.8%，或者经常达到从b浪的"价格"终点起扩展a*161.8%。
5) 设"a浪从起点到终点的比例"为"%a"，从a浪的终点开始，按照"%a"这个比例向外扩展，通常就是c浪的终点。

**平台形的时间指引**
1) b浪的时间通常介于a浪的时间的61.8%-161.8%之间。
2) c浪的时间介于a浪时间的61.8%和a浪、b浪中最短一浪的时间的161.8%之间。

### 5.2 平台形的定义

平台形是由a浪和b浪和c浪构成的、内部结构是3-3-5的、b浪必须大于等于a浪的70%的3段式调整浪。

**定义的几个要点**：
1) 3段式：而非5段式，这区别于驱动浪。
2) 内部结构是3-3-5：区别于单锯齿，同样三段式的单锯齿的内部结构是5-3-5。
3) b浪的运行总量必须大于等于a浪的运行总量的70%：平台形的b浪的运行总量既可以比a浪短，也可以比a浪长。最短等于a浪的运行总量的70%，最长等于a浪的运行总量的2倍。

### 5.3 平台形的内部结构

平台形的a浪必须是调整浪不能是三角形，通常是锯齿类即单锯齿、双锯齿、三锯齿之一，允许平台形、联合形。

平台形的b浪必须是调整浪不能是三角形，通常是锯齿类即单锯齿、双锯齿、三锯齿之一，允许联合形、平台形，但平台形极少。

平台形的c浪必须是推动浪或终结楔形，终结楔形极少，不能是引导楔形。

**内部结构常见共有5种**：
| 序号 | 结构 |
|------|------|
| 1 | 锯齿类+锯齿类+推动浪 |
| 2 | 锯齿类+联合形+推动浪 |
| 3 | 平台形+锯齿类+推动浪 |
| 4 | 平台形+联合形+推动浪 |
| 5 | 联合形+锯齿类+推动浪 |

**平台形的c-5浪不允许衰竭**：
- 当c浪是推动浪时不允许5浪衰竭。
- 当c浪是终结楔形时不允许5浪截尾。

### 5.4 平台形有三个子类及其外观

#### 5.4.1 子类一：规则平台形 (Regular flat)

b浪的运行总量小于等于a浪的运行总量，但必须大于等于a的运行总量的0.7。c浪超过或不超过a浪的终点，但接近a浪的终点，通常c浪的运行总量等于a浪（或b浪）的运行总量的0.618倍或1倍或1.236倍。外观看起来象个"N"。

#### 5.4.2 子类二：扩散平台形 (Expanded flat)

b浪的运行总量大于a浪的运行总量，但必须小于a浪的运行总量的2倍，通常b浪的运行总量等于a浪的运行总量的1.236倍。c浪超过a浪的终点，但必须小于a浪的3倍，通常c浪等于a浪（或b浪）的1.236倍到1.618倍。外观看起来象个"喇叭"。

扩散平台形通常出现于原有趋势较强（通常是刚经历了强劲的3浪后）、多空双方较为情绪化、斗争十分激烈的阶段。一方对另外一方进行强烈回撤，来回3次。

通常b浪的运行总量能接近a浪的运行总量的1.236倍，c浪的运行总量能接近a浪的运行总量的1.618倍。

#### 5.4.3 子类三：顺势平台形 (Running flat)

b浪的运行总量大于a浪的运行总量，但必须小于a浪的运行总量的2倍，通常b浪的运行总量等于a浪的运行总量的1.236倍。c浪不超过a浪的终点，但必须与a浪有重叠，且c浪的终点必须在a浪的终点和b浪的终点之间。外观看起来象个"倾斜的平台"。

**顺势平台形的特点**：
- 通常c浪的运行总量等于a浪（或b浪）的运行总量的0.618倍到1倍之间。外观看起来象个"菱形"。
- 顺势平台形出现的概率不高，通常出现于非常强劲的原有趋势的短暂休息阶段。
- 一个a浪的浅回撤后，原有趋势方向的b浪再次突进超过了a的起点，然后的c浪回撤力道很弱，仅仅切入a浪的起点，很快又被原有趋势快速阻截。
- 顺势平台形的前面或者后面经常是个延长浪。
- 顺势平台形以后，会沿着原有趋势继续运行，甚至更加强劲。因此交易者不可在顺势平台形以后逆势操作，应沿着原有趋势追击。
- 顺势平台形几乎不可能出现在2浪位置。在2浪阶段无需考虑顺势平台形的可能性。
- 即使在4浪阶段，尽量不主动假设一个顺势平台形。举例说，当发现"b浪"创新高以后，如果该"b浪"无法清晰的区分驱动浪还是调整浪，则尽量把它当作第5浪上升。除非该"b浪"是非常清晰确定的调整浪，再考虑这是一个顺势平台形。

### 5.5 平台形的a浪

平台形的a浪允许是平台形或联合形，平台形的b浪有可能大于a浪，即顺势平台形或扩散平台形，指引里说的"a浪中最大的子浪的价格通常小于a浪的价格"意思是通常出现规则平台形，不经常出现顺势平台形和扩散平台形。

平台形的a浪如果是单锯齿，其c浪等于b浪的情况通常不会出现。通常出现的是c浪大于b浪的情况。

**平台形a浪的时间**：a浪的最小时间没有限制。意思是最短可以是一根k线。

### 5.6 平台形的b浪与a浪的关系

b浪的运行总量必须大于等于a浪的运行总量的0.7倍且小于a浪的运行总量的2倍。

"平台形b浪必须回撤a浪的0.7倍以上"是平台形的重要特征，不满足这个条件一定不是平台形。相比较单锯齿，其b浪很少回撤a浪的0.7倍以上。

**平台形b浪的测算方法**：
- 测算b浪时，需要以a浪的最低点到a浪的最高点的价差作为基准，即a浪的运行总量。
- 平台形的a浪如果是平台形或联合形，其"起点到终点的价差即价格"可能是小于"最低点到最高点的价差即运行总量的"。
- 平台形的b浪以a浪的"运行总量"来测算，而不是以a浪的"价格"来做测算。

**平台形b浪的价格特征**：
- b浪中最大的子浪的价格通常小于a浪的价格，或者说a浪的价格通常大于b浪内所有子浪。
- b浪的"价格"通常大于a浪的"价格"的0.95倍，小于a浪的"价格"的1.4倍。

**平台形b浪的时间限制**：
- b浪的最小时间没有限制，最长时间不能超过a浪的时间的10倍。
- 如果b浪时间超过了a浪时间的10倍，很可能b浪在此之前已经结束，或者不是平台形的a浪和b浪而是三角形的a浪和b浪。
- b浪的时间通常介于a浪的时间的61.8%-161.8%之间。

### 5.7 平台形的c浪与a浪和b浪的关系

c浪必须与a浪有重叠。这规定了c浪的最小运动。规则平台形自然符合该规则，扩散平台形和顺势平台形的b浪会超过a浪起点，当c浪再次折返时必须切入a浪起点，即c浪与a浪有重叠。如果c浪和a浪没有重叠一定是数错浪了。

**c浪的运行总量限制**：
- c浪的"运行总量"必须小于等于a浪的"运行总量"和b浪的"运行总量"中最长的那个的2倍
- c浪的"运行总量"必须小于等于a浪的"价格"的3倍
- 上述两个比率是"且"的关系，即c浪的最大值是二者里取较小的
- 根据这条规则可以预测c浪最大值，即根据a浪和b浪的运行总量及价格，预测c浪的运行总量

**c浪的常见比率特征**：
- c浪经常超过b浪的1倍，即c浪超过b浪起点
- c浪不经常超过a浪的1.4倍
- c浪不经常超过b浪的1.4倍

**c浪的比率按照常见程度排序**：
- 第一常见：c=a*1（c浪价格等于a浪价格的1倍）
- 第二常见：c=a的终点（c浪终点等于a浪的终点）
- 第三常见：c=a*1.618（c浪价格等于a浪价格的1.618倍）。当c浪大于b浪时，这个比率很常见
- 第四常见：c浪=a的终点算起向外扩展0.618*a（c浪价格等于a浪终点起向外扩展a浪的0.618倍）。当c浪大于b浪时，这个比率很常见

**利用a浪价格运动百分比测算c浪终点**：
- 设"a浪从起点到终点的比例"为"%a"及价格运动百分比
- 从a浪的终点开始，按照"%a"这个价格运动百分比向外扩展，通常就是c浪的终点
- 可以用这个方法测算c浪

**c浪的时间限制**：
- c浪的时间不能超过a浪的时间的10倍且c浪的时间不能超过b浪的时间的10倍
- 即c浪时间不能超过a浪和b浪时间最短的那个的10倍
- 如果超过10倍时间可能不是平台形而可能是收缩三角形
- c浪的时间介于a浪时间的61.8%和a浪、b浪中最短一浪的时间的161.8%之间

### 5.8 平台形的艾略特通道

1) 当发现与之前驱动浪方向相反的第一个调整浪以后，即可以假设这是平台形的a浪
2) 当发现与这个假设的a浪方向相反的回撤是一个调整浪而且大于a浪的70%以后，即可以假设它是平台形的b浪
3) 画一个平行艾略特通道：前两个点选择0点和b点，第三个点选择a点
4) 如果c浪到达a浪的1倍，则正好到达这个平行艾略特通道的边缘。可以预先在c浪目标处布置交易指令

### 5.9 平台形的"价格"及"运行总量"

平台形的起点是其a浪的起点，终点是其c浪的终点。平台形的终点是下一个浪型的起点。

**平台形的"价格"**：
- 是该平台形的"a浪起点"和"c浪终点"之间的长度

**平台形的"运行总量"**：
- 是该平台形的所有子浪的最高点到所有子浪的最低点之间的长度

**特殊情况**：
- 平台形的"a浪起点"不一定是该平台形的最高点（或最低点），"c浪终点"也不一定是该平台形的最低点（或最高点）
- 比如扩散平台形或顺势平台形，此时平台形的"价格"仅是该平台形一部分，使平台形的"价格"和"运行总量"产生差异
- 如果不出现这种情况，平台形的"价格"和"运行总量"是等同的

**平台形各子浪的价格定义**：
- 平台形的a浪的价格是a浪的起点到a浪终点的价差
- b浪的价格是a浪的终点到b浪的终点的价差
- c浪的价格是b浪的终点到c浪的终点的价差

### 5.10 平台形会出现的位置

平台形分布十分广泛，下表是平台形可以出现的位置，其中下划线是常出现单锯齿的位置。

| 浪型 | 平台形可以出现的位置 |
|------|-------------------|
| 推动浪 | 2浪、4浪 |
| 53535型引导楔形 | 2浪、4浪 |
| 33333型引导楔形 | 2浪、4浪 |
| 终结楔形 | 2浪、4浪 |
| 单锯齿 | b浪 |
| 平台形 | a浪、b浪 |
| 三角形 | a浪、c浪、d浪 |
| 双锯齿 | x浪 |
| 三锯齿 | x浪、xx浪 |
| 双重横向整理 | w浪、x浪、y浪 |
| 三重横向整理 | w浪、x浪、y浪、xx浪、z浪 |

**平台形的出现频率**：
- 平台形经常出现在4浪位置，不经常出现在2浪位置

### 5.11 平台形的问答

**问：分辨平台形的诀窍是什么？**
答：70%。这个比例是诀窍。b浪的运行总量必须回撤a浪的运行总量的70%以上，如果没有达到这个比例，则不能假设为平台形。

**问：b浪回撤a浪的70%以上指的是a浪的什么的70%？**
答：指的是a浪的运行总量的70%以上，而不是a浪的价格，这是b浪的最小比率。按照指引，b浪的"价格"通常大于a浪的"价格"的0.95倍，小于a浪的"价格"的1.4倍。

**问：如何区分平台形的b浪和驱动浪？**
答：平台形的b浪是调整浪，驱动浪是5浪，这是最大的区别。平台形的b浪允许超过a浪的起点，看起来好像调整已完毕，开始新的沿着原有趋势的驱动浪了。但与a浪方向相反的那一段运动，如果只有3个子浪，而不是5个子浪，则它很可能是顺势平台形的b浪，接下来还有c浪的回撤。顺势平台形的b浪已经是非常强力了，对于c浪的回撤不宜看得太深。

**问：b浪如果回撤不超过a浪的70%，可能是平台形吗？**
答：不可能。这个比例是平台形的b浪的最低要求。最少要等于a浪的运行总量的70%。

**问：如何预见规则平台形、扩散平台形、顺势平台形？**
答：如果发现假设的b浪回撤超过了a浪的起点，它可能演化为扩散平台形或顺势平台形了。结合观察成交量。如果b的成交量比a的成交量放大不太多，而c的成交量缩小，它可能是顺势平台形。如果b的成交量比a的成交量放大较多，而c的成交量放大更多，它可能是扩散平台形。

**问：顺势平台形在4浪位置时如果c子浪未切入1浪但a子浪切入1浪可以吗？**
答：分为两类情况。第一类推动浪的第4浪时所有子浪都不允许切入1浪，虽然顺势平台形的c子浪未切入但a子浪有切入意味着有一个子浪切入了1浪，所以这是不允许的。第二类引导楔形或终结楔形的第4浪时，要求4浪必须切入1浪，所以这种情况下顺势平台的某个子浪切入1浪是允许的。

**问：平台形的c浪可以是收缩三角形吗？**
答：平台形的c浪绝不可能出现收缩三角形，只能是推动浪或者终结楔形，其中绝大多数情况是推动浪，终结楔形是极少数情况。如果发现c浪位置是收缩三角形，它很可能是双重横向整理且y浪是收缩三角形。

**问：收缩三角形可以当平台形的a浪吗？**
答：收缩三角形绝不可能充当平台形的a浪，只有锯齿类、平台形、联合形可以充当平台形的a浪，又以锯齿类最为常见，平台形、联合形不常见。

**问：有没有b浪没有超过a浪起点，而且c浪没有超过a浪终点的平台形？**
答：有。前提是b浪的运行总量超过a浪的运行总量的70%，且c浪必须是推动浪或终结楔形。

**问：顺势平台形或扩散平台形可以充当收缩三角形的a浪吗？**
答：可以，但非常少见。收缩三角形的a浪可以是锯齿类或平台形，通常是锯齿类，平台形很少见，即使平台形作为三角形的a浪也通常是规则平台形，顺势平台形或扩散平台形充当收缩三角形的a浪非常少见。

**问：平台形有没有c浪和a浪不重叠的情况？**
答：无论平台形是规则平台形、扩散平台形、顺势平台形，都不能出现c浪和a浪不重叠的情况。如果发现假设的平台形的c浪与a浪没有任何部分的重叠，则否定该平台形假设。

**问：平台形的a浪如何测算？**
答：当平台形出现在驱动浪后面时，a浪经常等于它"左侧的驱动浪"的0.382倍到0.5倍之间。而单锯齿的a浪等于它"左侧的驱动浪"的0.31倍到0.5倍之间。平台形出现在2浪时，它左侧的驱动浪是1浪；平台形出现在4浪时，它左侧的驱动浪是3浪；平台形出现在完整的5浪以后，它左侧的驱动浪是该"完整的5浪。"这个测算比率并非硬性规则，是常见现象，有助于在驱动浪以后立即假设平台形调整以及第一段a浪的回撤目标。

**问：为什么平台形会出现子浪的长度大于整个浪的长度？**
答：这个问题里的"长度"应该指的是"价格"，即平台形的a浪起点到c浪终点之间的价格差。平台形的c浪终点未必是该平台形的最低点/最高点，且b浪允许超过a浪的起点，则可能出现a浪起点到c浪终点之间的价差短于整个平台形的最高点到最低点之间的价差即运行总量，也可能短于其中的某个子浪。

**问：平台形的c浪特别长，达到a浪的2.618倍，是否可以？**
答：不可以。平台形的c浪最长只能到a浪的2倍。

**问：平台形最经常出现在什么位置？**
答：出现在4浪的频率高于出现在2浪的频率。

**问：顺势平台形能不能出现在2浪位置？**
答：平台形出现在2浪位置的频率就很少，在2浪位置出现顺势平台形的频率更低。尽量不考虑这样的情况。

**问：发现第一个平台形以后，它后续会如何发展？**
答：平台形后续的可能性较多，演化的可能性有（1）该平台形作为调整浪的全部，调整结束，重新延续原有的趋势开始新的驱动浪。（2）该平台形作为a浪，做更大级别的平台形，内部结构是3-3-5。（3）该平台形作为a浪，做更大级别的收缩三角形，内部结构是3-3-3-3-3。（4）该平台形作为w浪，向双重横向整理或三重横向整理演化。

**问：设"a浪从起点到终点的比例"为"%a"，从a浪的终点开始，按照"%a"这个比例向外扩展，通常就是c浪的终点。这句话是什么意思？**
答：举例说a浪起点是100，终点是95，起点到终点的比例即为5%。这个5%就是"%a"，从终点95开始向外扩展5%，即95*(1-5%)=90.25。
