## 三、黄金分割线算法（核心公式）

### 1. 基础定义
- **高点（H）**：当前趋势的最高点 = **5600**（第一浪终点）
- **低点（L）**：当前趋势的最低点 = 起点约**4300-4400**
- **涨幅（ΔP）**：ΔP = H - L ≈ **1200-1300**
- **回撤位（R）**：R = H - k × ΔP（k为黄金分割比率）
- **反弹位（B）**：B = L + k × ΔP（k为黄金分割比率）

### 2. 核心黄金分割比率

| 比率类型 | 常用比率值 | 应用场景 |
|---------|-----------|---------|
| **回撤比率** | 0.236 | 极浅回调（强势趋势） |
| | 0.382 | 浅回调（第二浪常见） |
| | 0.5 | 中性回调（关键情绪位） |
| | 0.618 | 深回调（标准第二浪） |
| | 0.786 | 极端回调（弱势第二浪） |
| | **0.8** | **蓝线位置（关键支撑）** |
| **反弹比率** | 0.382 | 弱反弹（调整初期） |
| | 0.5 | 中性反弹（关键压力位） |
| | 0.618 | 强反弹（调整末期） |
| **延伸比率** | 1.618 | 第三浪主升目标 |
| | 2.618 | 第三浪极端延伸目标 |

### 3. 计算公式模板

#### （1）第二浪回撤计算（基于第一浪5600）
```
已知：第一浪高点H = 5600，起点L ≈ 4300-4400，涨幅ΔP ≈ 1300
- 0.382回撤：5600 - 0.382 × 1300 ≈ 5103
- 0.5回撤：5600 - 0.5 × 1300 ≈ 4950
- 0.618回撤：5600 - 0.618 × 1300 ≈ 4797
- 0.786回撤：5600 - 0.786 × 1300 ≈ 4778
- 0.8回撤（蓝线）：4540（关键支撑）
```

#### （2）反弹压力计算（以当前低点为起点）
```
已知：低点L，高点H = 5600，涨幅ΔP = H - L
- 0.382反弹：L + 0.382 × ΔP
- 0.5反弹：L + 0.5 × ΔP
- 0.618反弹：L + 0.618 × ΔP
```

#### （3）第三浪延伸目标计算（假设第二浪结束）
```
已知：第一浪终点H_1 = 5600，第二浪低点L_2，涨幅ΔP_1 ≈ 1300
- 1.618倍延伸：L_2 + 1.618 × ΔP_1
- 2.618倍延伸：L_2 + 2.618 × ΔP_1
```
